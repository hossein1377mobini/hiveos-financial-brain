# صورت‌های مالی اساسی (Basic Financial Statements)

صورت‌های مالی اساسی، گزارش‌های کلیدی هستند که وضعیت مالی، عملکرد مالی و جریان‌های نقدی یک واحد تجاری را در یک دوره زمانی مشخص به تصویر می‌کشند. هدف اصلی این صورت‌ها، ارائه اطلاعات مفید به استفاده‌کنندگان (مانند سهامداران، بستانکاران، مدیران) برای تصمیم‌گیری‌های اقتصادی است. در ایران، تهیه و ارائه صورت‌های مالی عمدتاً بر اساس **استانداردهای حسابداری ایران** و نیز **قانون تجارت** صورت می‌گیرد.

## مفاهیم و اجزاء اصلی (Key Concepts and Components)

بر اساس **استاندارد حسابداری شماره 1 ایران (نحوه ارائه صورت‌های مالی)**، مجموعه کامل صورت‌های مالی شامل موارد زیر است:
1.  **ترازنامه (Statement of Financial Position / Balance Sheet)**
2.  **صورت سود و زیان (Statement of Profit or Loss / Income Statement)**
3.  **صورت سود و زیان جامع (Statement of Comprehensive Income)**
4.  **صورت تغییرات در حقوق صاحبان سرمایه (Statement of Changes in Equity)**
5.  **صورت جریان وجوه نقد (Statement of Cash Flows)**
6.  **یادداشت‌های توضیحی (Notes to Financial Statements)**

در این مبحث، تمرکز بر سه صورت مالی اصلی شامل ترازنامه، صورت سود و زیان و صورت جریان وجوه نقد است.

### 1. ترازنامه (Balance Sheet)

ترازنامه، یک **عکس لحظه‌ای** از وضعیت مالی شرکت در یک تاریخ معین (معمولاً پایان دوره مالی) است. این صورت نشان می‌دهد که یک واحد تجاری چه دارایی‌هایی (Assets) دارد، چه بدهی‌هایی (Liabilities) بر عهده دارد و حقوق صاحبان سرمایه (Equity) آن چقدر است.

**قاعده اساسی (Basic Rule):**
`دارایی‌ها = بدهی‌ها + حقوق صاحبان سرمایه`

**اجزای اصلی (Main Components):**
*   **دارایی‌ها (Assets):** منابع اقتصادی تحت کنترل واحد تجاری که انتظار می‌رود منافع اقتصادی آتی برای واحد تجاری داشته باشند.
    *   **دارایی‌های جاری (Current Assets):** دارایی‌هایی که انتظار می‌رود ظرف یک سال یا یک چرخه عملیاتی عادی (هر کدام طولانی‌تر است) به وجه نقد تبدیل شوند یا مصرف شوند (مانند موجودی نقد، حساب‌های دریافتنی، موجودی کالا).
    *   **دارایی‌های غیرجاری (Non-current Assets):** دارایی‌هایی که بلندمدت هستند و قصد نگهداری آنها برای بیش از یک سال وجود دارد (مانند دارایی‌های ثابت مشهود، سرمایه‌گذاری‌های بلندمدت).
*   **بدهی‌ها (Liabilities):** تعهدات فعلی واحد تجاری که تسویه آنها منجر به خروج منافع اقتصادی از واحد تجاری خواهد شد.
    *   **بدهی‌های جاری (Current Liabilities):** بدهی‌هایی که انتظار می‌رود ظرف یک سال یا یک چرخه عملیاتی عادی تسویه شوند (مانند حساب‌های پرداختنی، اسناد پرداختنی کوتاه مدت).
    *   **بدهی‌های غیرجاری (Non-current Liabilities):** بدهی‌های بلندمدت که سررسید آنها بیش از یک سال است (مانند وام‌های بلندمدت).
*   **حقوق صاحبان سرمایه (Equity):** باقیمانده دارایی‌ها پس از کسر بدهی‌ها. این بخش شامل سرمایه پرداخت شده، اندوخته‌ها، و سود (زیان) انباشته است.

**مثال فرضی از اقلام ترازنامه:**
| دارایی‌ها | مبلغ (ریال) | بدهی‌ها و حقوق صاحبان سرمایه | مبلغ (ریال) |
| :-------- | :----------- | :-------------------------- | :----------- |
| **دارایی‌های جاری:** | | **بدهی‌های جاری:** | |
| نقد و بانک | 50,000,000 | حساب‌های پرداختنی تجاری | 30,000,000 |
| حساب‌های دریافتنی | 70,000,000 | اسناد پرداختنی | 20,000,000 |
| موجودی کالا | 120,000,000 | **بدهی‌های غیرجاری:** | |
| **دارایی‌های غیرجاری:** | | وام بلندمدت | 80,000,000 |
| دارایی‌های ثابت مشهود | 200,000,000 | **حقوق صاحبان سرمایه:** | |
| (کسر استهلاک انباشته) | (40,000,000) | سرمایه | 200,000,000 |
| **جمع دارایی‌ها** | **400,000,000** | سود انباشته | 70,000,000 |
| | | **جمع بدهی‌ها و حقوق صاحبان سرمایه** | **400,000,000** |

**ارجاع قانونی:**
*   **ماده 232 قانون تجارت (مصوب 1347)**: "هیأت مدیره شرکت باید پس از انقضای سال مالی صورت دارائی و دیون شرکت را در پایان سال و همچنین ترازنامه و حساب عملکرد و حساب سود و زیان شرکت را به ضمیمه گزارش فعالیت و وضع عمومی شرکت طی سال مالی مزبور تنظیم کند."

### 2. صورت سود و زیان (Income Statement)

صورت سود و زیان، عملکرد مالی یک واحد تجاری را در یک **دوره زمانی مشخص** (مانند یک سال مالی یا یک فصل) نشان می‌دهد. این صورت شامل درآمدها (Revenues)، هزینه‌ها (Expenses) و در نهایت سود (Profit) یا زیان (Loss) خالص دوره است.

**قاعده اساسی (Basic Rule):**
`سود (زیان) خالص = درآمدها - هزینه‌ها`

**اجزای اصلی (Main Components):**
*   **درآمدهای عملیاتی (Operating Revenues):** درآمدهای حاصل از فعالیت‌های اصلی و مستمر شرکت (مانند فروش کالا یا ارائه خدمات).
*   **بهای تمام شده کالای فروش رفته (Cost of Goods Sold - COGS):** هزینه مستقیم کالاهایی که در طول دوره فروخته شده‌اند.
*   **سود (زیان) ناخالص (Gross Profit/Loss):** `درآمدهای عملیاتی - بهای تمام شده کالای فروش رفته`
*   **هزینه‌های عملیاتی (Operating Expenses):** هزینه‌های مرتبط با فعالیت‌های اصلی شرکت که مستقیماً به تولید کالا یا خدمات مربوط نیستند (مانند هزینه‌های اداری، عمومی و فروش، استهلاک).
*   **سود (زیان) عملیاتی (Operating Profit/Loss - EBIT):** `سود (زیان) ناخالص - هزینه‌های عملیاتی`
*   **درآمدها و هزینه‌های غیرعملیاتی (Non-operating Revenues and Expenses):** اقلامی که مستقیماً به فعالیت‌های اصلی شرکت مربوط نیستند (مانند درآمد اجاره، هزینه بهره).
*   **مالیات بر درآمد (Income Tax Expense):** مالیات محاسبه شده بر سود مشمول مالیات شرکت.
*   **سود (زیان) خالص (Net Profit/Loss):** رقم نهایی سود یا زیان پس از کسر تمامی هزینه‌ها و مالیات.

**مثال فرضی از صورت سود و زیان (اعداد به میلیون ریال):**
| شرح | سال 1404 | سال 1403 |
| :------------------------------ | :------- | :------- |
| فروش خالص (Net Sales) | 500 | 450 |
| (بهای تمام شده کالای فروش رفته) (Cost of Goods Sold) | (200) | (180) |
| **سود ناخالص (Gross Profit)** | **300** | **270** |
| (هزینه‌های اداری و عمومی) (Admin & General Expenses) | (50) | (45) |
| (هزینه‌های فروش) (Selling Expenses) | (30) | (25) |
| (استهلاک) (Depreciation) | (20) | (18) |
| **سود عملیاتی (Operating Profit)** | **200** | **182** |
| درآمد بهره (Interest Income) | 5 | 3 |
| (هزینه بهره) (Interest Expense) | (15) | (10) |
| **سود قبل از مالیات (Profit Before Tax)** | **190** | **175** |
| (مالیات بر درآمد) (Income Tax Expense) | (47.5) | (43.75) |
| **سود خالص (Net Profit)** | **142.5** | **131.25** |

**نکات کاربردی:**
*   **ماده 105 قانون مالیات‌های مستقیم:** نرخ مالیات بر درآمد شرکت‌ها و اشخاص حقوقی در ایران به طور عمومی 25% تعیین شده است (با توجه به آخرین اصلاحات و معافیت‌های احتمالی). در مثال بالا، مالیات 25% از سود قبل از مالیات محاسبه شده است.

### 3. صورت جریان وجوه نقد (Statement of Cash Flows)

صورت جریان وجوه نقد، تغییرات در وجه نقد و معادل‌های نقد یک واحد تجاری را طی یک **دوره زمانی مشخص** نشان می‌دهد. این صورت جریان‌های نقدی را بر اساس فعالیت‌های عملیاتی، سرمایه‌گذاری و تأمین مالی طبقه‌بندی می‌کند.

**استاندارد حسابداری شماره 2 ایران (صورت جریان وجوه نقد)**، استفاده از **روش مستقیم (Direct Method)** برای گزارشگری جریان‌های نقدی عملیاتی را ترجیح می‌دهد، اما روش غیرمستقیم (Indirect Method) نیز مجاز است.

**اجزای اصلی (Main Components):**
*   **جریان‌های نقدی حاصل از فعالیت‌های عملیاتی (Cash Flows from Operating Activities):** جریان‌های نقدی اصلی حاصل از فعالیت‌های درآمدزای شرکت (مانند دریافت‌های نقدی از مشتریان، پرداخت‌های نقدی به تأمین‌کنندگان و کارکنان).
*   **جریان‌های نقدی حاصل از فعالیت‌های سرمایه‌گذاری (Cash Flows from Investing Activities):** جریان‌های نقدی مربوط به تحصیل و واگذاری دارایی‌های بلندمدت و سایر سرمایه‌گذاری‌ها (مانند خرید و فروش دارایی‌های ثابت، سرمایه‌گذاری در سایر شرکت‌ها).
*   **جریان‌های نقدی حاصل از فعالیت‌های تأمین مالی (Cash Flows from Financing Activities):** جریان‌های نقدی مربوط به تغییرات در اندازه و ترکیب سرمایه و استقراض‌های شرکت (مانند دریافت وام، بازپرداخت وام، انتشار سهام، پرداخت سود سهام).
*   **خالص افزایش (کاهش) وجه نقد (Net Increase/Decrease in Cash):** جمع جریان‌های نقدی از سه فعالیت بالا.
*   **وجه نقد در ابتدای دوره (Cash at Beginning of Period):** موجودی نقد اول دوره.
*   **وجه نقد در پایان دوره (Cash at End of Period):** `خالص افزایش (کاهش) وجه نقد + وجه نقد در ابتدای دوره`.

**مثال فرضی از اقلام صورت جریان وجوه نقد (روش مستقیم، اعداد به میلیون ریال):**
| شرح | مبلغ (ریال) |
| :---------------------------------------- | :----------- |
| **جریان‌های نقدی حاصل از فعالیت‌های عملیاتی:** | |
| دریافت‌های نقدی از مشتریان | 480 |
| پرداخت‌های نقدی به تأمین‌کنندگان و کارکنان | (220) |
| پرداخت مالیات بر درآمد | (45) |
| **خالص وجه نقد حاصل از فعالیت‌های عملیاتی** | **215** |
| | |
| **جریان‌های نقدی حاصل از فعالیت‌های سرمایه‌گذاری:** | |
| خرید دارایی‌های ثابت مشهود | (30) |
| فروش دارایی‌های ثابت مشهود | 10 |
| **خالص وجه نقد حاصل از فعالیت‌های سرمایه‌گذاری** | **(20)** |
| | |
| **جریان‌های نقدی حاصل از فعالیت‌های تأمین مالی:** | |
| دریافت وام بلندمدت | 50 |
| بازپرداخت وام | (20) |
| پرداخت سود سهام | (10) |
| **خالص وجه نقد حاصل از فعالیت‌های تأمین مالی** | **20** |
| | |
| **خالص افزایش وجه نقد** | **215** |
| وجه نقد در ابتدای دوره | 50 |
| **وجه نقد در پایان دوره** | **265** |
